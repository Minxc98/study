### core 项目结构 ###

```
com.bage.demo.core

        -adapter // 适配、代理
        
        -config // 配置
        
            - ds // 数据源
            - prop // 属性
            - bean // bean注册

        -constant // 常量
        
        -mapping // 模型转化

        -gateway // 网关，程序入口
        
            - controller // 控制器 
            - mapping // 映射
            - service // 服务实现
            - job // 定时任务
            - mq // 消息中间件
            
        -model // 模型

        -repo // 数据库仓储实现
         - impl
            - mybatis
                - entity
                - mapper
                - mapping
            

        -logic // 最小业务单元逻辑
                
        -utils // 工具

```
